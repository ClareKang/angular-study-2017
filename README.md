#### Run Angular App Manually
```
$ (OPTIONAL) npm install -g http-server
$ http-server
```
OR
```
$ (OPTIONAL) npm install -g live-server
$ live-server
```

#### Create New Angular App With Angular/CLI
```
$ (OPTIONAL) npm install -g @angular/cli
$ ng new second-week 
$ cd second-week
```
